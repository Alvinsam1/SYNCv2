# SYNC WEBSITE MASTER BUILD SPECIFICATION

Build a premium cinematic storytelling website for the project called:

SYNC

using:

* Next.js (App Router)
* React
* TypeScript
* Tailwind CSS
* GSAP
* ScrollTrigger
* Framer Motion
* Lenis Smooth Scroll
* HTML Canvas image sequence rendering

The website should feel like a blend of:

* Apple Vision Pro website
* Apple AirPods Pro launch pages
* Nothing.tech
* Linear
* Arc Browser
* Modern AAA game reveal websites

The experience should feel futuristic, mysterious, cinematic, and premium.

The website is not a portfolio.

The website is the first chapter of a narrative universe.

Every animation should feel intentional and high-end.

PORTAL TRANSITION

IMPORTANT

When the user clicks a door while it is open:

The website should perform a cinematic transition.

NOT a standard page load.

NOT an abrupt route change.

Instead:

1. Door expands.
2. Camera pushes forward.
3. Doorway fills screen.
4. Interior light consumes viewport.
5. Entire website transitions into character timeline.

The doorway should feel like a portal.

The user should feel like they entered another reality.

Transition duration:

1.5–2.5 seconds.

Use:

GSAP timeline.

Scale.

Blur.

Light bloom.

Parallax.

Camera movement.

Motion blur.
